# README

Aplikacja Hejkamper to projekt demonstracyjny stworzony w celach edukacyjnych i prezentacyjnych. 
Nie oferuje rzeczywistych usług wynajmu kamperów. 

Podgląd: https://mysite-5r9a.onrender.com/

Aplikacja jest aktywnie rozwijana, więc jej logika / wygląd wciąż się zmienia.

### ✨ Główne Technologie

Aplikacja została zbudowana przy użyciu następujących kluczowych technologii:

* **Ruby on Rails:** Solidny szkielet webowy i silnik aplikacji.
* **Devise:** Kompleksowe zarządzanie uwierzytelnianiem i kontami użytkowników.
* **Shadcn/UI (z Tailwind CSS):** Nowoczesne, dostępnościowe komponenty UI dla lepszego doświadczenia użytkownika i szybkiego prototypowania.
* **Geocoder:** Moduł do geolokalizacji kamperów oraz wyszukiwania opartego na promieniu i odległości.
* **Image Processing (w tym Active Storage):** Obsługa, optymalizacja, miniaturyzacja i przechowywanie zdjęć kamperów.
